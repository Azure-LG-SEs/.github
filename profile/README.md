# 소개

LG를 지원하는 Solution Engineer들을 위한 GitHub 조직입니다.

이 조직은 LG 관련 기술 지원, PoC, 데모, 참조 아키텍처, 자동화 스크립트, 기술 문서 등을 협업하고 관리하기 위해 사용됩니다.

## 목적

- LG 지원을 위한 기술 자산 공유
- Solution Engineer 간 협업 공간 제공
- 데모, 샘플, 참조 구현 관리
- 아키텍처 패턴 및 기술 가이드 문서화
- 반복 업무 자동화를 위한 스크립트 관리

## 저장소 구성

이 조직의 저장소에는 다음과 같은 자료가 포함될 수 있습니다.

- 데모 애플리케이션
- 연동 및 통합 샘플
- 참조 아키텍처
- 자동화 스크립트
- 기술 문서 및 가이드
- 고객 지원을 위한 PoC 자료

## 기여 방법

각 저장소의 README와 기여 가이드를 확인한 후 작업해 주세요.

변경 사항은 Pull Request를 통해 리뷰 후 반영하는 것을 권장합니다.
